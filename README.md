# EC-Scrape

## Introduction
The purpose of this project is find EC numbers for proteins by performing a protein BLAST on gene sequences from a RAST genome annotation and using the results to search for the EC number of the protein in online databases such as NCBI and Uniprot.

## Setup
This project was developed and designed to work in windows 10 on a 64 bit machine.
